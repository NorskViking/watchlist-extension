# Welcome to the watchlist-extension project!

Welcome to this **hobby**-project of mine :smile: 

With this project, I aim to solve a small pain-point of mine, having a simple and easy way to keep track on all the hundreds of movies, TV series and anime I have watched, are watching, and are planning to watch in the future. Without having to manually update across several different sites. Especially when I want to keep track on what episode, of what season I was watching, before I got distracted with another series, or was away for a time.

## The use of AI in this project
I do not want to rely on an AI to write any code for me in this project.

But as this is also a solo hobby project, I have been using **claude.ai** as a discussion partner, to help me research different aspects around this project. 

After several rounds back and forth with **Claude**, to evaulaute things like what programming languages to potentially use, the base plan for this project, I gave it the task, to help me plan out and suggest for me, `milestones` and `issues` for the project. With strict instructions to not give me any code solutions, but only help me review my code when needed.

As warned, as I tasked **Claude** with creating the `issues` for this project, I take the issue description with a healthy dose of sceptism, and some of the issues will like mostly need to be revised, as I research and learn more about what is acctually possible to do in this type of project.
